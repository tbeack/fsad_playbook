# Claude Code Update Report — 2026-04-10

## New Version: v2.1.98 (released 2026-04-09)
## Previous State: None (first run)

---

## Categorized Changes

### New Features
- **Monitor tool** — New built-in tool for streaming events from background scripts
- **Google Vertex AI setup wizard** — Interactive wizard from login screen for GCP authentication, project/region config, credential verification, and model pinning
- **Amazon Bedrock setup wizard** — Interactive wizard from login screen for AWS authentication, region config, and model pinning (added in v2.1.92)
- **`CLAUDE_CODE_PERFORCE_MODE`** env var — Edit/Write/NotebookEdit fail on read-only files with `p4 edit` hint
- **Subprocess sandboxing** — PID namespace isolation on Linux via `CLAUDE_CODE_SUBPROCESS_ENV_SCRUB`; per-session script cap via `CLAUDE_CODE_SCRIPT_CAPS`
- **`--exclude-dynamic-system-prompt-sections`** flag — Print mode flag for cross-user prompt caching
- **`workspace.git_worktree`** in status line JSON — Set when in a linked git worktree
- **W3C `TRACEPARENT`** env var — Injected into Bash subprocesses when OTEL tracing is enabled
- **`/agents` tabbed UI** — Running tab shows live subagents; Library tab adds Run agent and View running instance actions
- **`/reload-plugins`** improved — Now picks up plugin-provided skills without restart

### Changed
- **`/claude-api` skill** — Updated to cover Managed Agents alongside Claude API
- **Default effort level** — Changed from medium to high for API-key, Bedrock/Vertex/Foundry, Team, and Enterprise users (v2.1.94)
- **LSP clientInfo** — Claude Code now identifies itself to language servers via `clientInfo`
- **Accept Edits mode** — Auto-approves filesystem commands prefixed with safe env vars or process wrappers

### Fixed (notable)
- Bash tool permission bypass with backslash-escaped flags
- Compound Bash commands bypassing forced permission prompts
- `--dangerously-skip-permissions` silently downgraded after writing to a protected path
- `/export` not honoring absolute paths and `~`
- `permissions.additionalDirectories` changes not applying mid-session
- Multiple `/resume` picker issues

### Removed
- `/tag` command (deprecated)
- `/vim` command (use `/config` → Editor mode instead)
