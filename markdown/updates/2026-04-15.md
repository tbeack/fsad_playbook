## Changelog Summary (v2.1.108 ŌĆō v2.1.109)

### v2.1.109
#### Improvements
- Extended-thinking indicator now shows a rotating progress hint (visual UX only)

### v2.1.108 (Major)
#### New Features
- **`/recap` command** ŌĆö replays session context when returning to a session; configurable in `/config`; manually invocable with `/recap`; force-enable with `CLAUDE_CODE_ENABLE_AWAY_SUMMARY=1` if telemetry is disabled
- **`ENABLE_PROMPT_CACHING_1H`** env var ŌĆö opt into 1-hour prompt cache TTL for API key, Bedrock, Vertex, and Foundry (`ENABLE_PROMPT_CACHING_1H_BEDROCK` deprecated); `FORCE_PROMPT_CACHING_5M` forces 5-minute TTL
- **Skill tool discovers built-in commands** ŌĆö the model can now discover and invoke built-in slash commands like `/init`, `/review`, and `/security-review` via the Skill tool
- **`/undo` is now an alias for `/rewind`** ŌĆö these two previously separate commands are now unified

#### Improvements
- `/model` now warns before switching models mid-conversation (next response re-reads full history uncached)
- `/resume` picker defaults to current-directory sessions; press `Ctrl+A` to show all projects
- Better error messages: rate limits distinguished from plan limits; 5xx/529 errors link to status.claude.com; unknown slash commands suggest closest match
- Added "verbose" indicator when viewing the detailed transcript (`Ctrl+O`)
- Warning shown at startup when `DISABLE_PROMPT_CACHING*` env vars are set

#### Fixes
- Multiple session, slash command, paste, and MCP bug fixes (see full CHANGELOG)
