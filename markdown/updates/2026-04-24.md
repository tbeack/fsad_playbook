# 2026-04-24 — Claude Code v2.1.119

## New Features
- `/config` settings (theme, editor mode, verbose, etc.) now persist to `~/.claude/settings.json` and participate in project/local/policy override precedence → **Affects:** Cheat Sheet (Configuration), Config Cascade
- Added `prUrlTemplate` setting to point the footer PR badge at a custom code-review URL → **Affects:** Config Cascade, Cheat Sheet CLI flags
- Added `CLAUDE_CODE_HIDE_CWD` environment variable to hide the working directory in the startup logo → **Affects:** Power Usage (Subprocess Sandboxing env vars)
- `--from-pr` now accepts GitLab merge-request, Bitbucket pull-request, and GitHub Enterprise PR URLs → **Affects:** Cheat Sheet CLI flags (Print/programmatic mode section)
- `--print` mode now honors the agent's `tools:` and `disallowedTools:` frontmatter — **Affects:** No playbook impact (internal behavior fix)
- `--agent <name>` now honors `permissionMode` for built-in agents — **Affects:** No playbook impact (minor behavioral alignment)
- PowerShell tool commands can now be auto-approved in permission mode — **Affects:** No playbook impact (Windows-specific)

## Changed

- Subagent and SDK MCP server reconfiguration now connects servers in parallel — **Affects:** No playbook impact (performance internal)
- Slash command suggestions now highlight matched characters — **Affects:** No playbook impact (UI refinement)
- Slash command picker now wraps long descriptions — **Affects:** No playbook impact (UI fix)
- OpenTelemetry: `tool_result` and `tool_decision` events now include `tool_use_id`; `tool_result` also includes `tool_input_size_bytes` → **Affects:** Monitoring (OTEL events table)
- Status line: stdin JSON now includes `effort.level` and `thinking.enabled` — **Affects:** No playbook impact (status line detail)
- Tool search disabled by default on Vertex AI; opt in with `ENABLE_TOOL_SEARCH` → **Affects:** Power Usage (Subprocess Sandboxing / env vars — Vertex AI note)
