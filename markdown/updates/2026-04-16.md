# 2026-04-16 — Claude Code v2.1.110

## New Features
- Added `/tui` command and `tui` setting — run `/tui fullscreen` to switch to flicker-free rendering in the same conversation → **Affects:** Cheat Sheet (new command, display/UI table or configuration table)
- Added `/focus` command — toggles focus view (previously this was bundled with `Ctrl+O`) → **Affects:** Cheat Sheet (new command, needs its own row)
- Added push notification tool — Claude can send mobile push notifications when Remote Control and "Push when Claude decides" config are enabled → **Affects:** Power Usage (minor; Remote Control section)
- Added `autoScrollEnabled` config to disable conversation auto-scroll in fullscreen mode → **Affects:** No playbook impact (config-only, niche fullscreen setting)
- Added option to show Claude's last response as commented context in the `Ctrl+G` external editor (enable via `/config`) → **Affects:** Cheat Sheet (update `Ctrl+G` description to mention this option)
- `--resume`/`--continue` now resurrects unexpired scheduled tasks → **Affects:** No playbook impact (internal behavior improvement)
- `/autocompact`, `/context`, `/exit`, and `/reload-plugins` now work from Remote Control → **Affects:** No playbook impact (remote compatibility expansion)
- Session recap now enabled for telemetry-disabled users (Bedrock, Vertex, Foundry); opt out via `/config` or `CLAUDE_CODE_ENABLE_AWAY_SUMMARY=0` → **Affects:** Cheat Sheet (update `/recap` row to mention opt-out env var)
