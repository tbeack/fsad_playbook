# 2026-04-28 ‚Äî Claude Code v2.1.120 + v2.1.121

State at last check: `lastVersion: v2.1.119`, `lastChecked: 2026-04-27`. Two new releases since then (v2.1.120 published in the v2.1.119 window, v2.1.121 published 2026-04-28). Combined release notes assessed below.

## New Features

### v2.1.121
- `alwaysLoad` option in MCP server config ‚Äî when `true`, the server's tools skip tool-search deferral and are always loaded. ‚Üí **Affects:** Power Usage (MCP / tool-search section)
- `claude plugin prune` subcommand + `plugin uninstall --prune` cascade ‚Äî removes orphaned auto-installed plugin dependencies. ‚Üí **Affects:** Cheat Sheet (plugin commands)
- `/skills` now has a type-to-filter search box for long lists. ‚Üí **Affects:** Cheat Sheet (`/skills` row ‚Äî minor note)
- PostToolUse hooks can now replace tool output via `hookSpecificOutput.updatedToolOutput` for **all** tools (previously MCP-only). ‚Üí **Affects:** Hooks section
- `--dangerously-skip-permissions` no longer prompts for writes to `.claude/skills/`, `.claude/agents/`, `.claude/commands/`. ‚Üí **Affects:** Permission Model / Skills (minor note)
- `/terminal-setup` now enables iTerm2 "Applications in terminal may access clipboard" so `/copy` works (incl. tmux). ‚Üí **Affects:** Cheat Sheet (`/terminal-setup` row note)
- `CLAUDE_CODE_FORK_SUBAGENT=1` now works in non-interactive sessions (SDK / `claude -p`). ‚Üí **Affects:** Subprocess Sandboxing env vars table
- OpenTelemetry: added `stop_reason`, `gen_ai.response.finish_reasons`, and `user_system_prompt` (gated behind `OTEL_LOG_USER_PROMPTS`) to LLM request spans. ‚Üí **Affects:** OTEL section
- LSP diagnostic summaries expand on click / Ctrl+O. ‚Üí **Affects:** No playbook impact (UX)
- VSCode: `/context` opens a native token usage dialog. ‚Üí **Affects:** No playbook impact (VSCode-specific UX)
- VSCode: voice dictation respects `accessibility.voice.speechLanguage`. ‚Üí **Affects:** No playbook impact

### v2.1.120
- **`claude ultrareview [target]`** non-interactive subcommand for CI/scripts ‚Äî `--json` for raw output, exit 0/1. ‚Üí **Affects:** Cheat Sheet (`/ultrareview` row, added in CBP-047) + a CLI subcommand note
- **`${CLAUDE_EFFORT}` skill template variable** ‚Äî skills can reference the current effort level inside their content. ‚Üí **Affects:** Building Skills section
- Windows: PowerShell fallback when Git Bash absent. ‚Üí **Affects:** No playbook impact (Windows install detail)
- `AI_AGENT` env var set for subprocesses so `gh` attributes traffic. ‚Üí **Affects:** Subprocess Sandboxing env vars table
- Spinner tips suppressed when desktop app / skills already exist. ‚Üí **Affects:** No playbook impact
- VSCode: `/usage` opens native Account & Usage dialog. ‚Üí **Affects:** No playbook impact (VSCode-specific)
- VSCode: voice dictation respects `language` setting in `~/.claude/settings.json`. ‚Üí **Affects:** No playbook impact
